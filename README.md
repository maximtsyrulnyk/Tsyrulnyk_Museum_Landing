Getting Started
Follow these steps to set up the project locally

Clone the repo
git clone https://github.com/maximtsyrulnyk/Tsyrulnyk_Museum_Landing.git
cd layout_landing-page
Install dependencies
npm install
# or
yarn install
Run the project locally
npm start
# or
yarn start
