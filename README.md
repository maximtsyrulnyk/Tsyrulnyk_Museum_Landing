🛠️ How to run locally

Clone the repository:

git clone https://github.com/maximtsyrulnyk/Tsyrulnyk_Museum_Landing.git

Go to project folder:

cd Tsyrulnyk_Museum_Landing

Install dependencies:

npm install

Run project:

npm start
